# Pokémon Champions - Análisis y Estrategia

Este repositorio contiene la base de datos de análisis estratégico, equipos y combates para el formato de **Pokémon Champions** (Singles y Dobles).

## 📂 Estructura del Repositorio

Tanto para la carpeta `Dobles` como para `Singles` (dentro de `M-B/`), se mantiene la siguiente estructura estandarizada:
- `My Team/`: Contiene los archivos individuales de los 6 Pokémon integrantes de la alineación activa.
- `Matches/`: Contiene los reportes analíticos de los combates jugados y preparación táctica.
- `OU/`: Contiene la base de datos de Pokémon de uso común de la temporada.

En la raíz del proyecto también se encuentran archivos de reglas generales:
- [reglas_pokemon_champions.md](./reglas_pokemon_champions.md): Reglas de construcción de equipos, restricciones de EVs y mecánicas permitidas.
- [referencia_tipos.md](./referencia_tipos.md): Tabla de efectividades específicas de tipos e inmunidades clave confirmadas en partida.

## 🤖 Guía para Agentes de IA y Analistas

Si eres un asistente de IA (como ChatGPT, Claude, Cursor, Windsurf, Copilot, etc.) o un analista estratégico, lee y sigue estrictamente las directrices del archivo [reglas_analisis_y_estructura.md](./reglas_analisis_y_estructura.md), además de los archivos de reglas y tipos indicados anteriormente, antes de proponer cambios de equipo, analizar combates o sugerir counters.
